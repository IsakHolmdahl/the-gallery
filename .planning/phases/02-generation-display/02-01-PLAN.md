# Plan: Phase 2 Generation & Display (Revised)

## Objective

Implement the AI generation pipeline that creates artwork from visitor prompts, displays it in an elegant frame, and plays museum docent narration. Revised to address cross-AI review feedback:
- **Sequential pipeline**: Image → Script → Voice (script is generated based on the actual image)
- Add serverless timeout protection (`maxDuration = 60`)
- Add autoplay fallback button for browser restrictions

## Context

### Phase Goal

AI generates artwork, displays in frame with docent narration

### Requirements
- CORE-04: OpenAI GPT-Image generates an image based on the prompt
- CORE-05: OpenAI GPT generates a museum docent voiceover script
- CORE-06: Eleven Labs TTS converts the script to voice narration
- CORE-07: AI-generated image appears in an elegant frame upon completion
- CORE-08: Voice narration plays automatically when artwork is revealed
- UI-02: Fancy decorative frame around generated image
- UI-03: Error states display friendly message with retry option
- TECH-01: OpenAI API integration for image generation
- TECH-02: OpenAI API integration for script generation
- TECH-03: Eleven Labs API integration for TTS voice generation

### Review Changes from Original Plan

**Changes made to address cross-AI review feedback:**

1. **Sequential pipeline (not parallel)** — Script is generated AFTER image completes so it can describe the actual artwork that was created.

2. **Add serverless timeout protection** — `export const maxDuration = 60;` at top of actions.ts prevents timeout on Vercel/serverless.

3. **Add autoplay fallback button** — ArtDisplay catches `audio.play()` rejection and shows "Play Narration" button if browser blocks autoplay.

### Existing Codebase
- Phase 1 completed with gallery UI (warm stone palette, shadcn components)
- Server action placeholder `createArt` in `src/app/actions.ts`
- Zod env validation already has: OPENAI_API_KEY, OPENAI_IMAGE_MODEL, OPENAI_SCRIPT_MODEL, ELEVEN_LABS_API_KEY, ELEVEN_LABS_VOICE_ID
- `.env.example` already configured

### Research Findings (from 02-RESEARCH.md)
- OpenAI image: `client.images.generate({ model: env.OPENAI_IMAGE_MODEL, prompt, n: 1, size: '1024x1024' })`
- OpenAI script: `client.chat.completions.create({ model: env.OPENAI_SCRIPT_MODEL, messages: [...] })`
- Eleven Labs: `elevenlabs.textToSpeech.stream(voiceId, { text, modelId: 'eleven_multilingual_v2' })`
- Pipeline: image → script → voice (sequential, script generated from actual image)

        async function collectStream(stream: ReadableStream<Uint8Array>): Promise<string> {
          const reader = stream.getReader();
          const chunks: Buffer[] = [];
          while (true) {
            const { done, value } = await reader.read();
            if (done) break;
            chunks.push(Buffer.from(value));
          }
          const buffer = Buffer.concat(chunks);
          return buffer.toString('base64');
        }

    2. Component with audio auto-play on mount using useEffect, WITH FALLBACK BUTTON:
       ```typescript
       'use client'
       export function ArtDisplay({ imageUrl, audioBase64, script, onReset }: ArtDisplayProps) {
         const audioRef = useRef<HTMLAudioElement>(null);
         const [showPlayButton, setShowPlayButton] = useState(false);

         useEffect(() => {
           if (audioRef.current) {
             audioRef.current.play()
               .then(() => setShowPlayButton(false))
               .catch(() => {
                 // Autoplay blocked - show fallback button
                 setShowPlayButton(true);
               });
           }
         }, [audioBase64]);

         const handlePlayNarration = () => {
           if (audioRef.current) {
             audioRef.current.play();
           }
         };

         return (
           <div className="space-y-6 animate-in fade-in slide-in-from-bottom-4 duration-700">
             {/* Gold museum frame */}
             <div className="relative p-6 bg-gradient-to-b from-amber-100 to-amber-50 rounded-lg shadow-2xl">
               {/* Inner mat (white border) */}
               <div className="p-3 bg-white rounded shadow-inner">
                 <img 
                   src={imageUrl} 
                   alt="Generated artwork" 
                   className="w-full h-auto rounded-sm"
                 />
               </div>
             </div>

             {/* Hidden audio player */}
             <audio 
               ref={audioRef}
               src={`data:audio/mp3;base64,${audioBase64}`}
             />

             {/* Fallback Play Narration button (shown if autoplay blocked) */}
             {showPlayButton && (
               <Button 
                 onClick={handlePlayNarration}
                 variant="default"
                 className="w-full"
               >
                 ▶ Play Narration
               </Button>
             )}

             {/* Create another button */}
             <Button 
               onClick={onReset}
               variant="outline"
               className="w-full"
             >
               Create Another
             </Button>
           </div>
         );
       }
       ```
