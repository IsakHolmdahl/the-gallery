## Objective

Add a 1-second delay to voice narration so it doesn't start immediately when the image is revealed. The image has a 700ms fade-in animation — the voice should begin after the image is fully visible.

**Purpose:** Voice starting at the exact same moment as the image reveal feels jarring. A 1-second pause lets the visitor absorb the artwork before the docent narration begins.

**Output:** `setTimeout` wrapper around `audioRef.current.play()` in the mount effect.

## Context

@src/components/art-display.tsx

Current state:
- `useEffect` fires `audioRef.current.play()` immediately on mount (line 24)
- Image has a `transition-all duration-700 ease-out` fade-in (line 41)
- Voice and image animation start simultaneously

## Tasks

### Task 1: Add 1-second delay to audio playback

**Files:** `src/components/art-display.tsx`

**Action:** Wrap the `audioRef.current.play()` call in a `setTimeout(..., 1000)` inside the existing `useEffect`. Add cleanup `return () => clearTimeout(timer)` to prevent firing after unmount.

**Done:** Voice starts 1 second after image reveal. Timer cleaned up on unmount.
